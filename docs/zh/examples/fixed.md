# 固定
- `config.HEIGHT`高度，高度为 0 表示自适应，根据`config.OFFSET_HEIGHT`调整
- `config.MAX_HEIGHT`最大高度，高度为 0 表示自适应，根据`config.OFFSET_HEIGHT`调整
- fixed可为`"left","right"`
-
