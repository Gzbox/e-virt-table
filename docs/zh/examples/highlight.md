# 高亮


## Config

| 参数                 | 说明           | 类型    | 可选值 | 默认值 |
| -------------------- | -------------- | ------- | ------ | ------ |
| HIGHLIGHT_HOVER_ROW | 启用悬浮高亮 | boolean | —      | false   |
| HIGHLIGHT_HOVER_ROW_COLOR | 悬浮颜色 | string | —      | rgba(186,203,231,0.1)   |
