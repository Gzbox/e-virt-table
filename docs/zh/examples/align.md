# 对齐

## Column

| 参数          | 说明     | 类型   | 可选值                          | 默认值 |
| ------------- | -------- | ------ | ------------------------------- | ------ |
| align         | 水平对齐 | string | `"left"`, `"center"`, `"right"` | center |
| verticalAlign | 垂直对齐 | string | `"top", "middle", "bottom"`     | middle |

## 水平对齐

- 默认 `"center"`
- align 可为`"left"`, `"center"`, `"right"`
