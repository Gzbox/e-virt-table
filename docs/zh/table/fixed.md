# 固定

## Column

| 参数             | 说明                   | 类型    | 可选值              | 默认值 |
| ---------------- | ---------------------- | ------- | ------------------- | ------ |
| fixed            | 是否固定列             | string  | `"left"`, `"right"` | 100    |


## 固定表头

- `config.HEIGHT`高度，高度为 0 表示自适应

::: demo

fixed/header
h:350px
:::

## 固定表头(最大高度)

- `config.MAX_HEIGHT`高度，高度为 0 表示自适应

::: demo

fixed/header1
h:400px
:::

## 固定列

- fixed 可为`"left","right"`

::: demo

fixed/left-right
h:320px
:::
